Para iniciar el juego hay que ejecutar el archivo .jar que 
hay en la carpeta "Server".
Si no funciona clicando dos veces en el archivo, iremos a la cmd de 
Windows, llegaremos a la ruta en la que se encuentra el archivo .jar
y ejecutaremos el código
java -jar API_Rest-0.0.1-SNAPSHOT.jar
y con ello pondremos el server a correr en el puerto 8080.
Una vez arrancado el server, podemos jugar al juego
buscando en Google las IPs que indican el equipo en local:
http://127.0.0.1:8080/
http://localhost:8080/
Y si estás en un ared local:
http://ip-del-eqipo-servidor:8080/
